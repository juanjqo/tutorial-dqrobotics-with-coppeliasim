Tutorial: DQ Robotics with CoppeliaSim#

Get started


Master the DQ Robotics library with CoppeliaSim!

Learn the basic concepts of CoppeliaSim with your favorite library for dual quaternion algebra.

Use your favorite language!

Focused for Matlab. However, a lot of examples for Python and C++ are also provided.

Multiplatform

Although is mainly oriented to Ubuntu LTS, there are instructions for Windows and macOS users.

If you’re looking for the DQ Robotics documentation, this is NOT it. For the DQ Robotics documentation, refer to this link.
